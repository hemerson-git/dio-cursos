Error Validation by types

The target is to create a function that receive an array and return it again if its length matches the number in the second parameter in the function, otherwise an error must to be thrown.

  • Create a function that takes an array and a number.
  • Makes the following validations:
    • If params not to be sent, throw an *ReferenceError*
    • If the Array is ot nof type "object", throw an error of type: TypeError.
    • If the number is not of type "number", throw an error of type: TypeError.
    • If the arrays length is not equals to the number sent to the function throw a RangeError.
  • Use try...catch expression
  • Filter catch calls by each type of error by using the instance of operator.
